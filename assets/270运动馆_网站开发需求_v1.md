# 270运动馆品牌官网 - 开发需求 v1.0

## 一、项目定位
**交付目标**：一个可以直接成交给客户（270运动馆）的完整品牌官网，含前端用户端 + 管理后台，具备完整的会员体系与商业化能力。

**品牌名称**：270运动馆 / BEAUTY CYCLE 270
**运营主体**：福州坤成体育发展有限公司
**品牌定位**：女性专属健身服务品牌
**核心价值**：安全、私密、无评判、高适配
**slogan**：BEAUTY CYCLE 270 · 她的运动美学
**品牌调性**：温柔而有力量，女性友好，精致专业，社群感

## 二、品牌关键数据（用于官网展示）
- 创始人：徐宁
- 2022年创立于福州
- 2026年初完成种子轮融资，投后估值500万
- 超1000名核心会员，月度续费率68%
- 累计服务10W+人次女性
- 计划在福州核心商圈及社区新增3-5家门店

## 三、战略方向
1. **个性化训练计划 + 营养建议**的定制化服务
2. **健身 + 社交 + 健康管理**的多元生态
3. **私域电商**：向会员提供优质健康食品与运动用具

## 四、设计风格规范

### 配色方案（女性友好、高端精致）
- **主色**：玫瑰粉/豆沙粉系（如 #E8A0B5 或 #D4859B）
- **背景色**：米白/奶杏色（如 #FDF8F5 或 #FAF5F2）
- **文字色**：深棕/深灰（如 #3A2E2A 或 #4A4A4A）
- **点缀色**：珊瑚橘（活力感）或 薄荷绿（清新感）
- **整体感觉**：高端瑜伽馆/普拉提工作室气质，非传统硬核健身房

### 视觉元素
- 柔和曲线、圆润边角
- 渐变光斑、柔和阴影
- 女性运动剪影/插画
- 纤细优雅的无衬线字体

## 五、功能模块总览

### 5.1 登录与注册系统（必做）
- 手机号 + 验证码登录
- 微信登录（前端预留入口）
- 新用户注册完善资料（身高、体重、运动目标、健身基础）
- 忘记密码/重置密码
- 退出登录
- Token鉴权机制，未登录访问会员页面自动跳转登录

### 5.2 会员/VIP体系（必做）
- **会员等级**：普通会员 / 银卡会员 / 金卡会员 / 钻石会员
- **会员权益展示**：各等级权益对比表
- **会员卡购买页面**：月卡/季卡/年卡选项 + 价格
- **会员到期提醒**：个人中心显示到期时间
- **会员专享内容**：金卡以上可看专属课程、专属折扣
- **积分系统**：签到/上课/消费获得积分，积分可兑换课程或商品

### 5.3 课程预约系统
- 课程列表（按分类/时间/教练筛选）
- 课程详情页（介绍、教练、时间、地点、剩余名额）
- 在线预约/取消预约
- 我的课表
- 课程签到

### 5.4 私教服务
- 私教团队展示页
- 私教详情（资质、擅长领域、学员评价）
- 私教课程购买
- 私教课预约

## 六、各页面详细需求

### 前端用户端（移动端优先）

#### 1. 首页 `/`
- 顶部：品牌Logo + 搜索 + 消息通知
- Banner轮播：品牌slogan、促销活动、新课推荐
- 快捷入口：预约课程、私教服务、会员中心、商城
- 品牌故事区块：关于270 + 核心数据展示（1000+会员、68%续费率等）
- 热门课程推荐：卡片式展示，点击进入详情
- 教练团队：头图+姓名+擅长，点击查看详情
- 会员故事/学员蜕变：前后对比 + 感言
- 底部导航：首页、课程、商城、我的

#### 2. 课程页 `/courses`
- 分类Tab：全部 / 普拉提 / 瑜伽 / 女性力量 / 孕产修复 / 体态管理
- 课程列表：封面图、名称、教练、时间、地点、价格、剩余名额
- 筛选功能：按时间、按难度、按教练
- 课程详情页：图文介绍、课程安排、适合人群、教练介绍、学员评价、预约按钮

#### 3. 商城页 `/mall`
- 分类：运动服饰、健康食品、瑜伽辅具、护肤美容
- 商品列表：图片、名称、价格、会员价标签
- 商品详情页：图文详情、规格选择、加入购物车、立即购买
- 购物车
- 订单管理（我的订单）
- 收货地址管理

#### 4. 饮食/营养页 `/diet`
- 定位：女性营养管理
- 今日营养摄入概览（环形图：热量/蛋白质/碳水/脂肪）
- 饮食记录功能
- AI营养建议（保留现有功能）
- 推荐食谱（减脂期/备孕期/产后等）

#### 5. 我的/个人中心 `/profile`
- 头像、昵称、会员等级标识
- 核心数据：累计运动次数、消耗卡路里、连续打卡天数
- 功能菜单：
  - 我的课程（已预约/历史）
  - 我的订单（商城/课程）
  - 我的收藏
  - 会员中心（升级会员、查看权益）
  - 我的积分
  - 身体数据（身高体重体脂等记录）
  - 设置（个人信息、账号安全、通知设置、关于我们）

#### 6. 登录/注册页 `/login`
- 手机号输入 + 验证码获取
- 登录/注册一键切换
- 微信登录按钮
- 用户协议与隐私政策链接

#### 7. 会员中心页 `/vip`
- 当前会员等级与到期时间
- 会员权益列表
- 各等级对比
- 会员卡购买选项（月卡/季卡/年卡）
- 立即开通/升级按钮

### AI中台（中端）
- 配色调整匹配品牌风格
- 数据总览：用户数、活跃度、课程完成数、AI建议数
- AI智能洞察：活跃度预警、课程完成率分析、营养建议优化
- 用户分层分析：高/中/低活跃、沉默用户占比
- 会员数据：会员等级分布、续费率分析、收入趋势
- 7日趋势图表、系统日志

### 管理后台（后端）
- 配色调整匹配品牌风格
- **仪表盘**：核心数据概览
- **用户管理**：用户列表、查看/编辑/禁用、会员等级调整
- **课程管理**：课程CRUD、排期管理、预约管理
- **教练管理**：教练信息管理、课程分配
- **订单管理**：商城订单、课程订单、会员订单
- **商品管理**：商品CRUD、分类管理、库存管理
- **财务管理**：收入统计、退款处理
- **内容管理**：Banner管理、公告管理、文章管理
- 筛选/导出/新增功能按钮
