# 270运动馆 - 设计风格重大调整 v2.0

## 核心问题（必须解决）
当前设计太像小程序/App界面，缺少网页的高级感和品牌调性。需要全面重做视觉风格。

## 核心方向
**苹果官网风格 + 包装袋品牌色 = 极简高级的运动生活方式品牌站**

参考：apple.com 、lululemon官网 、Alo Yoga官网

---

## 一、品牌色（严格按包装袋）
- **品牌主色（橙）**：#C45A2C 或 #D4612F（陶土橙/焦糖橙，参考包装袋上的270颜色），不是亮橙不是珊瑚橘，是偏深偏暖的橙棕色
- **背景色**：纯白色 #FFFFFF（大面积留白，主体是白的）
- **深色文字**：#1D1D1F（苹果黑，接近纯黑但不是纯黑）
- **次要文字**：#86868B（苹果灰）
- **分割线**：#D2D2D7 或 #F5F5F7（极浅灰）
- **辅助点缀色**：可以用深灰色系 + 品牌橙做点缀，不要粉色系！不要粉色！不要粉色！

**重要**：去掉所有粉色、玫瑰色、豆沙色元素。主色调是：白 + 黑灰 + 品牌橙（点缀）。参考包装袋的配色感觉。

---

## 二、整体设计语言（苹果风）

### 1. 大量留白
- 页面左右边距要大（桌面端两侧留白10%+）
- 区块之间间距要大，不要挤在一起
- 内容区域呼吸感十足，不要铺满

### 2. 大字号排版
- 页面大标题：非常大（48px-80px），粗体，字间距收紧
- 副标题/小标题：中等大小，清晰有力
- 正文：适中，行高宽松（1.5-1.6）
- 字体层级对比强烈，一眼能看出主次
- 字体用系统无衬线字体（SF Pro / PingFang SC 风格），不要用衬线字体

### 3. 极简卡片与边框
- 卡片用极细边框或无框（阴影也尽量淡）
- 圆角统一，中等大小（不要太圆也不要太方）
- 按钮设计简洁，边框按钮/填充按钮两种

### 4. 图片使用
- 大图展示，沉浸式
- 图片质量要高，生活方式感
- 图片可以全出血（边缘到屏幕边）
- 图片和文字穿插排版，有杂志感

### 5. 动效
- 滚动时元素优雅淡入
- 悬停效果微妙不夸张
- 页面过渡流畅自然
- 不要花哨动效，克制有质感

## 三、首页重新设计（重点中的重点）

按苹果官网 + lululemon官网的感觉来做

### 区块1：Hero首屏（全屏沉浸式）
- 全屏高质量大图背景（女性运动/生活方式图，自然光，高级感）
- 图片上叠加半透明黑色渐变（底部深上面透明）
- 居中或左对齐的大标题：
  - 超大字号「270」
  - 「BEAUTY CYCLE」
  - 副标题一句话
- 一个简洁的CTA按钮（白色填充或描边）
- 向下滚动提示

### 区块2：品牌标语/文案区
- 大面积留白
- 居中的一段品牌宣言，字号较大
- 类似苹果官网"Our Story"那种感觉

### 区块3：核心课程展示
- 大标题「课程」
- 2-3列大卡片布局，每张卡片是大图 + 课程名
- 卡片悬停有微妙放大效果
- 参考苹果产品展示的方式

### 区块4：品牌数据/亮点
- 横向排列 4 个数字：1000+会员 / 68%续费率 / 10W+人次 / 2022创立
- 数字很大，标签文字很小
- 极简风格，白色背景

### 区块5：教练团队
- 大标题「我们的教练」
- 横向排列，大图头像 + 姓名 + 简介
- 干净利落

### 区块6：会员体系
- 一整块品牌橙或深灰背景的区块
- 大标题 + 会员权益介绍
- 对比苹果官网的产品介绍区块

### 区块7：CTA转化区
- 大面积留白居中
- 一句话 + 一个大按钮
- 「预约体验课」

### 区块8：页脚
- 极简风格，浅灰背景
- 链接分组清晰
- 底部版权信息

## 四、各页面风格调整要求

### 课程页
- 顶部大标题 + 简短介绍
- 课程卡片大一些，图片为主
- 筛选栏简洁
- 不要小程序那种密密麻麻的感觉

### 商城页
- 参考 lululemon 商城
- 大图商品展示
- 干净的产品网格
- 价格和产品名清晰，不要太多装饰

### 个人中心
- 简洁干净
- 会员卡片可以有设计感但不要花哨
- 功能列表用简洁的列表样式

### 登录页
- 全屏背景图 + 简洁登录框
- 苹果风的极简表单
- 不要毛玻璃花哨效果

### 管理后台 & AI中台
- 也同步调整为白 + 黑灰 + 橙点缀的风格
- 简洁专业的数据看板
- 参考苹果设计语言的后台感觉

---

## 五、必须改掉的问题
1. ❌ 去掉所有粉色、玫瑰色、豆沙色 → 改为白 + 黑灰 + 品牌橙
2. ❌ 去掉小程序感的密集布局 → 加大留白，拉开间距
3. ❌ 去掉花哨的渐变、光晕、玻璃拟态 → 极简、干净、克制
4. ❌ 去掉可爱/柔和的圆角过度 → 中等圆角，专业感
5. ❌ 去掉衬线字体标题 → 用现代无衬线字体
6. ✅ 增加大图展示、大标题、大片留白
7. ✅ 增加网页的纵深感和呼吸感
8. ✅ 品牌橙只用在关键按钮、logo、强调元素上，作为点缀

---

## 六、实现优先级
1. **先改全局配色和字体系统**（影响所有页面）
2. **重做首页**（最重要的门面）
3. **调整其他主要页面的布局和留白**
4. **最后处理后台和中台的风格统一**

功能保留不变，只改视觉设计和布局排版。
